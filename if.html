Explications

Dans le chapitre précédent, nous avons appris comment comparer des données. En tant que tel, ce n’est pas très utile dans un programme. Toutefois, lorsque ces expressions sont utilisées avec le mot-clé if (si), elles deviennent très utiles.

Ce mot-clé permet d’exécuter une série d’instructions, seulement si une condition est remplie (si elle est évaluée à true).

Il existe un deuxième mot-clé else (sinon), qui permet de définir un autre bloc de code à exécuter, seulement si la condition n’est pas remplie.

Syntaxiquement, la condition est placée entre parenthèses et la série d’instructions est placée juste après, entre deux accolades ({}).

Par exemple : 

if(true) {
  //ce code sera exécuté
} else {
  //ce code ne sera pas exécuté
}

Il est aussi possible d’avoir d’autres conditions avec if avant le else final. Ces conditions supplémentaires sont dénotées avec les mots-clés else if.1 Par exemple : 

if(false) {
  //ce code ne sera pas exécuté
} else if(true) {
  //ce code sera exécuté
} else {
  //ce code ne sera pas exécuté
}

Si aucune des conditions n’est remplie, c’est le code sous le else qui sera exécuté. Par exemple : 

if(false) {
  //ce code ne sera pas exécuté
} else if(false) {
  //ce code ne sera pas exécuté
} else {
  //ce code sera exécuté
}

Évidemment, les conditions ne seront pratiquement jamais simplement des valeurs booléennes. Ces valeurs sont uniquement employées à titre d’exemple. Dans la vraie vie, tu écrirais plutôt des expressions conditionnelles avec des variables. Par exemple : 

function estPositif(n) {
  if(n > 0) {
    return “Ce nombre est positif”;
  } else if (n < 0) {
    return “Ce nombre est négatif”;
  } else {
    return “Ce nombre est zéro”;
  }
}

Exercices

Complète la fonction estPair(n). Cette fonction doit prendre pour entrée un nombre n et retourner (return) une valeur booléenne. Cette valeur booléenne doit être true si n est pair et false si n est impair. Par exemple, estPair(5) retourne false et estPair(8) retourne true. Si tu éprouves des difficultés, retourne voir le chapitre sur les opérations.


Exécuter

Modifie ta fonction estPair(n) pour qu'elle retourne true seulement si n est pair et positif.
