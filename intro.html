Introduction

Avant de commencer à programmer, il faudrait définir ce qu’est un programme.

Un programme est comme une recette pour un ordinateur. C’est une liste d’instruction qu’il suit à la lettre, afin de transformer les données entrées par l’utilisateur. Tout comme une recette indique à un humain comment transformer des ingrédients afin d’arriver à un résultat final.

La seule différence entre un programme et une recette : une recette est écrite par un humain pour des humains, alors qu’un programme est écrit par un humain pour des ordinateurs. En écrivant une recette, on peut donc être assez imprécis, puisque la personne qui la lira sera au courant de certaines conventions.

Si j’écris « Dans un petit bol, battre l’œuf avec le lait », un humain comprend qu’il doit utiliser une fourchette ou un batteur, et remuer rapidement le mélange de lait et d’oeufs. Même si le mot « battre » a plusieurs sens, l’humain comprend grâce au contexte.

Ce n’est pas le cas d’un ordinateur. Il ne comprend pas le contexte ou les conventions humaines. Il faut donc être très précis. Pour qu’une machine comprenne, on pourrait écrire l’exemple de tout à l’heure de la manière suivante : 

                      Déplacer “oeuf” dans “bol”;
Déplacer “lait” dans “bol”;
Remuer contenu de “bol” avec “fourchette” à vitesse “élevée”;
Répéter instruction #3 25 fois;

Évidemment, ce n’est pas un vrai programme. C’est ce qu’on appelle du pseudo-code. C’est-à-dire que le code n’est pas écrit dans un véritable langage de programmation. Un langage de programmation est ce qu’on utilise pour communiquer avec l’ordinateur. Il en existe plusieurs avec différents champs d’applications.1

Celui que nous allons utiliser au cours de ce tutoriel est JavaScript. Il est principalement utilisé pour le développement web. C’est-à-dire pour la création de sites internet. Il peut toutefois être utilisé partout : pour créer des jeux vidéos, pour créer des applications mobiles… C’est pourquoi j’ai décidé d’utiliser ce langage pour ce tutoriel : il est très versatile.2

Dans le chapitre suivant, nous explorerons le concept des variables. Mais avant, comme il est coutume dans les tutoriels de programmation, commençons par le programme de base par excellence : « Hello, world! ».
