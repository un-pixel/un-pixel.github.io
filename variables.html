Déclaration d'une variable

Explications

Tout comme en mathématiques, le concept de « variables » est très présent en programmation. Une variable sert à désigner une donnée (nombre, texte... ), tout comme ton nom permet de te désigner. Par exemple, je pourrais assigner la valeur ‘25 octobre 2002’ à la variable dateDeNaissance.

Remarque la manière dont est notée la variable dateDeNaissance. C'est la manière standard de nommer les variables en Javascript. Cette écriture se nomme camelCase. Il existe toutefois d'autres standards comme snake_case (date_de_naissance). Au final ça a peu d'importance. L'important est de rester cohérent et de garder en tête que Javascript est sensible aux majuscules et aux accents. La variable DateDeNaissance est donc tout à fait différente de la variable dateDeNaissance.

En JavaScript, ce processus est assez simple. Il suffit d’écrire let dateDeNaissance = “25 octobre 2002”;. Le mot-clé let peut te sembler abstrait. C’est normal. En mathématiques, il suffit d’écrire, par exemple, x = 7. C’est aussi possible en Javascript : ce mot-clé peut ne pas être mis. Par contre, nous verrons un peu plus tard que c’est généralement une mauvaise idée.

Attention, lorsque la valeur d'une variable est sous forme de texte, il faut le déclarer en utilisant les guillemets anglais (""). Une suite de caractères sans guillemets fait plutôt référence à une autre variable.

Une autre différence avec les mathématiques est qu’une variable peut changer de valeur. En effet, une fois la variable déclarée avec let, il est possible de modifier sa valeur, en lui assignant une nouvelle donnée, sans mot-clé. Par exemple : 

                        let x = 8;
//présentement la valeur de x est de 8
x = 7;
//maintenant la valeur de x est de 7

Toutefois, dans certains cas il est préférable d’empêcher qu’une variable puisse changer de valeur. C’est dans ces cas que le mot-clé const est utilisé. Par exemple, si je déclare const x = 8;, la valeur de x ne pourra plus être altérée. x = 7 produirait simplement une erreur.

Un autre mot-clé existe pour déclarer une variable : var. Celui-ci est toutefois de moins en moins utilisé, remplacé par let, qui a essentiellement la même fonction.1
